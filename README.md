# ASCONA Scanner · Возвраты Android

Flutter-приложение для ускорения обработки возвратов в зоне Маркетплейс.

## Что уже заложено в MVP

- Сканирование кода возврата/заказа и кода товара через камеру.
- Ручной ввод, если камера или штрихкод не читается.
- Компактный режим: поставщик и состояние сохраняются для следующих позиций.
- Журнал позиций с возможностью исправления без повторного сканирования.
- Локальное хранение данных на устройстве.
- Экспорт CSV и отправка через системное меню Android.
- Аналитика смены: всего, принято, проблемы.

## Как собрать APK на GitHub

1. Загрузить папку `ascona_returns_android` в GitHub-репозиторий.
2. Убедиться, что файл `.github/workflows/build_android.yml` лежит в корне репозитория.
3. Открыть вкладку `Actions`.
4. Запустить workflow `Build Android APK`.
5. Скачать артефакт `ascona-returns-apk`.

## Как собрать локально

На ПК должен быть установлен Flutter SDK и Android SDK.

```powershell
flutter pub get
flutter create --platforms=android .
flutter build apk --release
```

Готовый APK будет здесь:

```text
build/app/outputs/flutter-apk/app-release.apk
```
